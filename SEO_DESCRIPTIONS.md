# Hello Beauty Lounge - SEO Descriptions Guide

## Overview
This document contains optimized meta descriptions for all pages of the Hello Beauty Lounge website. Each description is crafted to:
- Stay within 50-160 characters (Google's display limit)
- Include primary keywords for the page
- Include a compelling call-to-action
- Improve click-through rates from search results

## Best Practices Applied

✓ **Character Count:** All descriptions are 50-160 characters
✓ **Keywords:** Each includes 3-5 relevant keywords
✓ **Location:** All include "Red Deer" or location-specific terms
✓ **Action-Oriented:** Include CTAs where appropriate (Book, Learn, Discover)
✓ **Unique:** Each description is unique to the page content
✓ **Compelling:** Written to encourage clicks from search results
✓ **Brand Consistency:** Maintains Hello Beauty Lounge brand voice

## Maintenance & Updates

Review and update descriptions:

- **Performance:** A/B test descriptions that underperform
